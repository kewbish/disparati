# disparati

Solutions to [Fly.io](https://fly.io)'s [Gossip Glomers](https://fly.io/dist-sys/) (distributed systems) challenges. 'Disparati' is a portmanteau of 'distributed apparati'.  
Written in Go, May 2023 to present.
